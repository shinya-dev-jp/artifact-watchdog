# Security

`artifact-watchdog` is designed to run locally and read local files selected by the user.

Please do not publish reports or config files that include:

- secrets or tokens;
- private filesystem paths;
- account IDs;
- customer or employee data;
- internal project names;
- unreleased business metrics.

If you find a security issue in the tool itself, please open a minimal public issue that describes the behavior without including private data.
